
# go-context
Sample project to understand context usage in Golang

## run

### value
```shell script
cd value
go run main.go
```

### middleware
```shell script
cd middleware/server
go run main.go

cd middleware/client
go run main.go
```

### cancellation
```shell script
cd cancellation/server
go run main.go

cd cancellation/client
go run main.go
```

### timeout
```shell script
cd timeout/server
go run main.go

cd timeout/client
go run main.go
```
